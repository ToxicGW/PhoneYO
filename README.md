# 📱 PhoneYO Sales Company - E-Commerce Front-End Application

A clean, highly responsive front-end user interface for a modern smartphone retail shop. This project focuses on client-side state management, responsive grid layouts, and dynamic DOM manipulation without relying on external frameworks.

## 🏢 About PhoneYO Sales Company
PhoneYO Sales Company is a one-stop destination for the latest smartphones from leading brands such as Samsung, Apple, Oppo and Xiaomi. The platform is designed to offer a diverse range of models to suit every need and budget from high-end flagship devices to reliable mid-range devices. 

Our core mission is to provide customers with exclusive discounts, a wide variety of items, and a smooth dynamic online shopping experience.


## 🚀 Technical Stack & Tooling
*   **Frontend:** HTML5, CSS3 (Custom Grid & Flexbox layouts)
*   **Core Logic:** JavaScript (Vanilla ES6+ for state tracking)
*   **Version Control:** Git / GitHub Workflow
*   **Open Source License:** MIT License

## 💡 Key Engineering Features
*   **Dynamic Cart Management:** Implemented pure JavaScript array state management to handle adding, removing, and dynamically updating smartphone quantities in real-time.
*   **Mobile-First Responsive Layout:** Utilized fluid CSS Grid layouts and media queries to ensure an optimal shopping experience across diverse mobile, tablet, and desktop viewports.
*   **Interactive Product UI:** Integrated native JavaScript event listeners for clean views of smartphone Quantities with the selected items and after clicking the "Calculate" button it will give the total price in the same page .


## 📁 Repository Structure
*   `/index.html` - Main landing page and store template.
*   `/Online_Shopping_Form.html` - Modular checkout module handling customer form inputs.
*   `/Contact_Page.html` - Store contact hub displaying guide and structural user inquiry boxes.
*   `/info.html` - Dedicated documentation page outlining company distribution details and store policies.
*   `/CssStyles.css` - Custom styling sheet containing layout rules and responsive breakpoints.
*   `/Scripts.js` - Core functional architecture managing the cart engine and UI rendering loops.
